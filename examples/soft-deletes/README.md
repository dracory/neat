# Soft Deletes Example

This example demonstrates how to use soft deletes for records.

## Features Demonstrated

- Adding soft delete capability to models
- Soft deleting records
- Including soft-deleted records with WithTrashed()
- Querying only soft-deleted records with OnlyTrashed()
- Excluding soft-deleted records (default behavior)
- Restoring soft-deleted records
- Force deleting (permanent deletion)

This example is a placeholder. Full implementation will be added once the soft delete API is finalized.
