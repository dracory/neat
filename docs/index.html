Neat ORM Documentation

A powerful and elegant ORM library for Go with Laravel Eloquent-inspired syntax

🚀 Key Advantage: Complete Migration System

Most Go ORMs lack comprehensive schema migration support. Neat ORM includes a complete migration system with schema builder, rollback support, and automatic tracking - something most competitors either lack entirely or require third-party tools for.
